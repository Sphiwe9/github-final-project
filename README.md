# Simple Interest Calculator 

## Descrition
This project is a simple interest 
calculator script created as part of a 
micro-finance startup initiative. It
helps users calculate interest on loans 
easily.

## Features
- Calculate simple interest
- Easy to use
- Lightweight bash script

## How to Use
Run the script and enter:
- Principal amount
- Rate of interest
- Time period

- ## Formula
Simple Interest = (Principal * Rate * Time) /
100
